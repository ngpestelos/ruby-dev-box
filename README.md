## Ruby Development Box

A [Vagrant](http://vagrantup.com) box for [Ruby](http://ruby-lang.org) projects.

### Usage

    git clone git@github.com:ngpestelos/ruby-dev-box.git
    cd ruby-dev-box
    vagrant up
